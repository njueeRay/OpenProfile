# Agent 视觉规格文档

> **文档状态：** ✅ v6.3.0 正式确立  
> **负责人：** Profile Designer × Dev 协同  
> **用途：** 为 `/agents` 页面、博文作者标记、未来 Pixel Avatar 提供统一视觉语言基准

---

## 视觉语言双轨

| 轨道 | 用途 | 格式 |
|------|------|------|
| **几何符号**（Symbol）| 站点内部 `/agents` 卡片、博文 byline | Unicode 字符（可在 Markdown / HTML 中直接使用）|
| **像素头像**（Pixel Avatar）| 对外传播层（Discussion 封面、社交媒体）| PNG 64×64 / 256×256（v6.x 后续阶段实现）|

---

## 7 个 Agent 完整视觉定义

### Brain · 战略协调

| 维度 | 值 | 语义 |
|------|-----|------|
| **主色** | `#3B5BDB`（深蓝）| 深度思考 + 可信赖的权威感 |
| **Symbol** | `⬡`（空心六边形）| 六边形多维综合思考，连接多方的结构节点 |
| **Pixel Avatar 风格** | 深蓝底 + 白色六边形线框 | 待实现 |
| **角色素描** | 团队唯一汇报窗口，将用户的模糊意图精确化为可执行指令；在所有人开始工作之前，Brain 先想清楚「为什么做」。 |

## 配色原则

1. **对比度**：所有颜色在 `#0d1117`（GitHub Dark）背景上对比度 ≥ WCAG AA（4.5:1）
2. **差异化**：7 色覆盖蓝/绿/紫/橙/红/金/青，视觉上无混淆
3. **品牌锚点**：站点主强调色 `#58a6ff`（GitHub Blue）保留给 UI 链接态，Agent 色系与其不冲突
4. **暗色优先**：所有色值针对暗色背景调优，浅色模式通过透明度自适应
